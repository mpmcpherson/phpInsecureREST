Object Collections in PHP

Saturday, September 17, 2011

When I first started learning object-oriented PHP and I needed to access objects in an array, I used to store them in...well...arrays. This was back in the days of PHP 4, when object support was finally starting to get somewhere, and well before the addition of the ArrayIterator class. All was fine for awhile, but eventually I grew tired of doing things like this to find a particular object:

  1. <?php
  3. foreach($arrayOfUsers as $user) {
  4. if ($user->getUsername() == 'spammy') {
  5. // do stuff here
  6. reset($arrayOfUsers);
  7. break;
  8. }
  9. }

I began running into situations where I needed to remove one of the objects from the array, but I wanted to somehow mark the object as deleted so that I could, for example, pass it to a saveArray() method of a DataMapper object, and let that handle the object deletion. Or perhaps I needed to sort an array of Users alphabetically by the users' first names when it wasn't convenient (or possible) to do that with the DBMS. Also, what happens when you serialize() an array of objects? The array's internal pointer is lost. On the rare occasions I needed to store an array in mid-loop with its internal pointer intact, I was just out of luck.

While looking for a solution to these problems, I kept running into the term "Collection" here and there, but it seemed to mean different things depending on which programming language was being utilized. All of the PHP solutions I found were at least a little "icky" smelling. Finally I came up with this Collection class which I still use it to this day. I feel the syntax is easier/clearer than PHP's ArrayIterator class, and unless I were to write an ArrayIterator wrapper class I would lose many of the methods I have come to rely on.

The Collection Class

  1. <?php
  4. class Collection
  5. {
  7. protected $objects; // array
  8. protected $deletedObjects; // array
  9. protected $resetFlag;
  10. protected $numObjects;
  11. protected $iterateNum;
  14. public function __construct()
  15. {
  16. $this->resetIterator();
  17. $this->numObjects = 0;
  18. $this->objects = array();
  19. $this->deletedObjects = array();
  20. }
  23. public function add($obj)
  24. {
  25. $this->objects[] = $obj;
  26. $this->numObjects++;
  27. }
  30. public function next()
  31. {
  32. $num = ($this->currentObjIsLast()) ? 0 : $this->iterateNum + 1;
  33. $this->iterateNum = $num;
  34. }
  37. public function isOdd()
  38. {
  39. return $this->iterateNum%2==1;
  40. }
  43. public function isEven()
  44. {
  45. return $this->iterateNum%2==0;
  46. }
  48. /*
  49. get an obj based on one of it's properties.
  50. i.e. a User obj with the property 'username' and a value of 'someUser'
  51. can be retrieved by Collection::getByProperty('username', 'someUser')
  52. -- assumes that the obj has a getter method
  53. with the same spelling as the property, i.e. getUsername()
  54. */
  55. public function getByProperty($propertyName, $property)
  56. {
  57. $methodName = "get".ucwords($propertyName);
  59. foreach ($this->objects as $key => $obj) {
  60. if ($obj->{$methodName}() == $property) {
  61. return $this->objects[$key];
  62. }
  63. }
  64. return false;
  65. }
  67. /*
  68. alias for getByProperty()
  69. */
  70. public function findByProperty($propertyName, $property)
  71. {
  72. return $this->getByProperty($propertyName, $property);
  73. }
  76. /*
  77. get an objects number based on one of it's properties.
  78. i.e. a User obj with the property 'username' and a value of 'someUser'
  79. can be retrieved by Collection::getByProperty('username', 'someUser')
  80. -- assumes that the obj has a getter method
  81. with the same spelling as the property, i.e. getUsername()
  82. */
  83. public function getObjNumByProperty($propertyName, $property)
  84. {
  85. $methodName = "get".ucwords($propertyName);
  87. foreach ($this->objects as $key => $obj) {
  88. if ($obj->{$methodName}() == $property) {
  89. return $key;
  90. }
  91. }
  92. return false;
  93. }
  95. /*
  96. get the number of objects that have a property
  97. with a value matches the given value
  98. i.e. if there are objs with a property of 'verified' set to 1
  99. the number of these objects can be retrieved by:
  100. Collection::getNumObjectsWithProperty('verified', 1)
  101. -- assumes that the obj has a getter method
  102. with the same spelling as the property, i.e. getUsername()
  103. */
  104. public function getNumObjectsWithProperty($propertyName, $value)
  105. {
  106. $methodName = "get".ucwords($propertyName);
  107. $count = 0;
  108. foreach ($this->objects as $key => $obj) {
  109. if ($obj->{$methodName}() == $value) {
  110. $count++;
  111. }
  112. }
  113. return $count;;
  114. }
  116. /*
  117. remove an obj based on one of it's properties.
  118. i.e. a User obj with the property 'username' and a value of 'someUser'
  119. can be removed by Collection::removeByProperty('username', 'someUser')
  120. -- assumes that the obj has a getter method
  121. with the same spelling as the property, i.e. getUsername()
  122. */
  123. public function removeByProperty($propertyName, $property)
  124. {
  125. $methodName = "get".ucwords($propertyName);
  127. foreach ($this->objects as $key => $obj) {
  128. if ($obj->{$methodName}() == $property) {
  129. $this->deletedObjects[] = $this->objects[$key];
  130. unset($this->objects[$key]);
  131. // reindex array & subtract 1 from numObjects
  132. $this->objects = array_values($this->objects);
  133. $this->numObjects--;
  134. $this->iterateNum = ($this->iterateNum >= 0) ? $this->iterateNum - 1 : 0;
  135. return true;
  136. }
  137. }
  138. return false;
  139. }
  142. public function currentObjIsFirst()
  143. {
  144. return ($this->iterateNum == 0);
  145. }
  148. public function currentObjIsLast()
  149. {
  150. return (($this->numObjects-1) == $this->iterateNum);
  151. }
  154. public function getObjNum($num)
  155. {
  156. return (isset($this->objects[$num])) ? $this->objects[$num] : false;
  157. }
  160. public function getLast()
  161. {
  162. return $this->objects[$this->numObjects-1];
  163. }
  166. public function removeCurrent()
  167. {
  168. $this->deletedObjects[] = $this->objects[$this->iterateNum];
  169. unset($this->objects[$this->iterateNum]);
  170. // reindex array & subtract 1 from iterator
  171. $this->objects = array_values($this->objects);
  173. if ($this->iterateNum == 0) { // if deleting 1st object
  174. $this->resetFlag = true;
  175. } elseif ($this->iterateNum > 0) {
  176. $this->iterateNum--;
  177. } else {
  178. $this->iterateNum = 0;
  179. }
  180. $this->numObjects--;
  181. }
  184. public function removeLast()
  185. {
  186. $this->deletedObjects[] = $this->objects[$this->numObjects-1];
  187. unset($this->objects[$this->numObjects-1]);
  188. $this->objects = array_values($this->objects);
  189. // if iterate num is set to last object
  190. if ($this->iterateNum == $this->numObjects-1) {
  191. $this->resetIterator();
  192. }
  193. $this->numObjects--;
  194. }
  197. public function removeAll()
  198. {
  199. $this->deletedObjects = array_merge($this->deletedObjects, $this->objects);
  200. $this->objects = array();
  201. $this->numObjects = 0;
  202. }
  204. /*
  205. sort the objects by the value of each objects property
  207. $type:
  208. r regular, ascending
  209. rr regular, descending'
  210. n numeric, ascending
  211. nr numeric, descending
  212. s string, ascending
  213. sr string, descending
  214. */
  215. public function sortByProperty($propName, $type='r')
  216. {
  217. $tempArray = array();
  218. $newObjects = array();
  220. while ($obj = $this->iterate()) {
  221. $tempArray[] = call_user_func(array($obj, 'get'.ucwords($propName)));
  222. }
  224. switch($type)
  225. {
  226. case 'r':
  227. asort($tempArray);
  228. break;
  229. case 'rr':
  230. arsort($tempArray);
  231. break;
  232. case 'n':
  233. asort($tempArray, SORT_NUMERIC);
  234. break;
  235. case 'nr':
  236. arsort($tempArray, SORT_NUMERIC);
  237. break;
  238. case 's':
  239. asort($tempArray, SORT_STRING);
  240. break;
  241. case 'sr':
  242. arsort($tempArray, SORT_STRING);
  243. break;
  244. default:
  245. throw new General_Exception(
  246. 'Collection->sortByProperty():
  247. illegal sort type "'.$type.'"'
  248. );
  249. }
  251. foreach ($tempArray as $key => $val) {
  252. $newObjects[] = $this->objects[$key];
  253. }
  254. $this->objects = $newObjects;
  255. }
  258. public function isEmpty()
  259. {
  260. return ($this->numObjects == 0);
  261. }
  264. public function getCurrent()
  265. {
  266. return $this->objects[$this->iterateNum];
  267. }
  270. public function setCurrent($obj)
  271. {
  272. $this->objects[$this->iterateNum] = $obj;
  273. }
  276. public function getObjectByIterateNum($iterateNum)
  277. {
  278. return (
  279. isset($this->objects[$iterateNum])
  280. ? $this->objects[$iterateNum]
  281. : false
  282. );
  283. }
  286. public function iterate()
  287. {
  288. if ($this->iterateNum < 0) {
  289. $this->iterateNum = 0;
  290. }
  291. if ($this->resetFlag) {
  292. $this->resetFlag = false;
  293. } else {
  294. $this->iterateNum++;
  295. }
  296. if ( $this->iterateNum == $this->numObjects
  297. || !isset($this->objects[$this->iterateNum])
  298. ) {
  299. $this->resetIterator();
  300. return false;
  301. }
  303. return $this->getCurrent();
  304. }
  307. public function resetIterator()
  308. {
  309. $this->iterateNum = 0;
  310. $this->resetFlag = true;
  311. }
  314. public function __toString()
  315. {
  316. $str = '';
  317. foreach ($this->objects as $obj) {
  318. $str .= '--------------------------<br />'.$obj.'<br />';
  319. }
  320. return $str;
  321. }
  324. #################### GETTERS
  326. public function getDeletedObjects()
  327. {
  328. return $this->deletedObjects;
  329. }
  331. public function getIterateNum()
  332. {
  333. return $this->iterateNum;
  334. }
  336. public function getNumObjects()
  337. {
  338. return $this->numObjects;
  339. }
  341. #################### SETTERS
  343. public function setDeletedObjects($key, $val)
  344. {
  345. $this->deletedObjects[$key] = $val;
  346. }
  348. public function resetDeletedObjects()
  349. {
  350. $this->deletedObjects = array();
  351. }
  353. }

An important note before we look at implementation: I use getters/setters to access the properties of all of my objects (properties are set as protected or private). So if I have a Foo class with a property named bar, I would access it through $foo->getBar();. Getters and setters are not everyone's cup of tea; without getting into that debate you can feel free to modify the methods above that call object properties through getters.

Alright, so what can we do with this thing? First let's create an example object, and populate a Collection with 10 of those suckers:

  1. <?php
  3. class Example
  4. {
  5. protected $id;
  7. public function setId($id) { $this->id = $id; }
  9. public function getId() { return $this->id; }
  10. }
  12. // create a new Collection class
  13. $collection = new Collection;
  15. // populate it with 10 "Example" objects
  16. for($i=1; $i <= 10; $i++) {
  18. $example = new Example;
  19. $example->setId($i);
  21. // add the object to the collection
  22. $collection->add($example);
  23. }
